5. Configuração do Servidor Proxy (Squid)

  • Software Utilizado: Squid
  • Porta do Proxy: 3128

  • Regras de Controle de Acesso (ACLs):

  • Rede Permitida: 10.0.0.0/24

  • Sites Bloqueados: /etc/squid/bloqueados.txt (domínios bloqueados)

Exemplo de configuração no arquivo /etc/squid/squid.conf:

http_port 3128

acl rede_local src 10.0.0.0/24

http_access allow rede_local

acl sites_bloqueados dstdomain "/etc/squid/bloqueados.txt"

http_access deny sites_bloqueados

  • Logs do Squid: /var/log/squid/access.log (para monitoramento de acessos)
